# CodeGuardian Frontend

Basit Next.js frontend uygulaması - JavaScript kod güvenlik analizi için.

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
npm run dev
```

Uygulama [http://localhost:3000](http://localhost:3000) adresinde çalışacaktır.

## Backend

Backend'in `http://localhost:8080` adresinde çalışıyor olması gerekmektedir.

## Özellikler

- JavaScript kod girişi
- Kod analizi
- Güvenlik sorunlarının listelenmesi
- Sorun detayları (severity, açıklama, öneriler)
